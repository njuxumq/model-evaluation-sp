# 流程感知说明

本文档定义模型评测技能的流程感知和进度展示规范。

---

## 核心原则

| 原则 | 说明 |
|------|------|
| **渐进式披露** | 默认输出简化信息，完整信息按需展示 |
| **情境感知** | 根据用户行为动态调整输出详细程度 |
| **路径可回溯** | 展示"已走过的路径"帮助用户理解上下文 |

---

## 展示层级

| 时机 | 展示内容 |
|------|----------|
| 进入 SKILL | 全流程概览（阶段列表 + 每阶段任务列表） |
| 进入阶段 | 阶段路径 + 当前阶段任务列表 |
| 进入任务 | 任务路径 + 当前任务步骤列表 |
| 进入步骤 | 步骤路径 + 分支判断/步骤描述 |
| 进入流程 | 流程路径 + 流程内步骤列表 |

---

## 状态符号

| 符号 | 状态 |
|------|------|
| `✓` | 已完成 |
| `●` | 当前 |
| `○` | 待执行 |
| `~` | 已跳过 |

---

## 展示模板

### 进入 SKILL 时

```
📍 模型评测流程概览

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
阶段1: 初始化

任务1: 环境检测 → 任务2: 鉴权验证 → 任务3: 会话目录

阶段2: 构建配置

任务1: 确认评测场景 → 任务2: 确认评测标准 → 任务3: 确认评委配置

阶段3: 评测集处理

任务1: 处理评测集

阶段4: 执行评测

任务1: 提交评测任务 → 任务2: 查询任务状态 → 任务3: 展示评测结果
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

                             │      │

## 分支判断格式

### 展示判断条件

```
{判断条件}
  ├─ {选项A} → {后果A}
  ├─ {选项B} → {后果B}
  └─ {选项C} → {后果C}
```

### 展示判断结果

```
📋 判断结果：{结果} → {后续动作}
```
