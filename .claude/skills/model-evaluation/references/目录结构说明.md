## 工作目录

```
{work-dir}/.eval/
├── env.cfg                     # 环境配置缓存
├── auth.json                   # 鉴权Token
└── session-{id}/               # 会话目录（session-{8位字母数字}）
    ├── evalset/                # 评测集相关文件
    │   ├── evalset-prepared.{ext}     # 待标准化评测集
    │   ├── evalset-structure.json     # 字段结构分析
    │   ├── evalset-fields-mapping.json # 字段映射
    │   ├── evalset-status.json        # 状态检测结果
    │   ├── evalset-standard.jsonl     # 标准化评测集
    │   └── evalset-meta.json          # 上传后元数据
    ├── evaltask/               # 任务相关文件
    │   ├── evaltask-meta.json         # 任务元信息
    │   └── evaltask-result.json       # 评测结果
    ├── selected-models.json    # 选定模型列表
    ├── eval-dimension.json     # 维度配置
    └── eval-judge.json         # 评委配置
```

## 只读目录说明

### assets/ 目录

**内容**：内置模板资源

**规则**：
- ❌ 不可修改
- ✅ 可复制到工作目录后修改

**自定义模板位置**：`{work-dir}/.eval/{session-id}/`

### scripts/ 目录

**内容**：可执行脚本

**规则**：
- ❌ 不可修改脚本内容
- ✅ 通过命令行参数传递配置

**参数格式**：
```bash
{python-env}{python-cmd} {script-path} {subcommand} --arg1 value1 --arg2 value2
```

## 变量说明

| 变量 | 说明 | 使用场景 | 示例值 |
|------|------|----------|--------|
| `{skill-dir}` | 技能安装目录 | 全局 | `/home/user/.claude/skills/model-evaluation` |
| `{work-dir}` | 当前工作目录 | 全局 | `/home/user/projects/my-project` |
| `{session-id}` | 会话目录名 | 全局 | `session-a3c8f4ci` |
| `{python-cmd}` | Python 命令 | 全局 | `python` 或 `python3` |
| `{python-env}` | 环境变量前缀 | 全局 | `PYTHONUTF8=1 ` 或空 |
| `{ext}` | 文件扩展名 | 评测集处理 | `jsonl`, `csv`, `xlsx` |
| `{state_token}` | OAuth 状态令牌 | 鉴权流程 | UUID 格式 |
| `{auth_code}` | OAuth 授权码 | 鉴权流程 | 用户授权后获得 |
| `{task_id}` | 评测任务 ID | 执行评测 | 平台返回的任务标识 |
| `{platform_url}` | 在线报告链接 | 执行评测 | 评测结果展示地址 |
