# 进度展示详细规范

本文档定义阶段/任务/步骤/流程级的进度展示模板。

---

## 进入阶段时

```
📍 当前位置：{阶段名称}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
初始化 ✓ → 构建配置 ✓ → {当前阶段} ● → 执行评测 ○
                             │
                             ├─ 任务1: {名称} ○
                             └─ 任务2: {名称} ○
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
                             │      │

```

## 分支判断格式

**展示判断条件**：

```
{判断条件}
  ├─ {选项A} → {后果A}
  ├─ {选项B} → {后果B}
  └─ {选项C} → {后果C}
```

**展示判断结果**：

```
📋 判断结果：{结果} → {后续动作}
```
