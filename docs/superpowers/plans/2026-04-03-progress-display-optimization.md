```
#### 状态符号

| 符号 | 状态 |
|------|------|
| `✓` | 已完成 |
| `●` | 当前 |
| `○` | 待执行 |
| `~` | 已跳过 |
```

```

**展示判断结果**：

```
📋 判断结果：{结果} → {后续动作}
