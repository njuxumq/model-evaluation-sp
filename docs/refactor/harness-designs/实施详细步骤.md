## 四、回滚方案

| 回滚点 | 回滚方式 | 适用情况 |
|--------|----------|----------|
| 阶段1-6 | git checkout | 单个文件创建失败 |
| 阶段7-9 | git checkout --branch | 批量重构失败 |
| 全量回滚 | git reset --hard | 校验失败，无法修复 |

**回滚命令**：
```bash
# 单文件回滚
git checkout -- {file-path}

# 批量回滚
git checkout -- .

# 全量回滚
git reset --hard HEAD
```

## 五、风险预案

| 风险 | 影响 | 预案 |
|------|------|------|
| 知识文件内容不完整 | 校验失败 | 对照原文件补充 |
| 引用路径错误 | 流程中断 | 逐个检查引用 |
| 收集器逻辑复杂 | 实现困难 | 拆分为更小单元 |
| 用户交互设计不合理 | 用户体验差 | 迭代优化 |
